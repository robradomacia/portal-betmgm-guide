# portal-betmgm-guide

## 项目简介

`portal-betmgm-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面主要面向特定场景下的信息展示与功能引导，提供清晰、轻量的前端页面资源。

本仓库不针对任何特定网站或域名，仅作为 HTML 文件的集中存储与版本管理之用。

## 目录结构

```
portal-betmgm-guide/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 静态资源（CSS、JS、图片等）
├── README.md       # 本文件
└── .gitignore      # Git 忽略配置
```

- `pages/`：每个子目录或文件代表一个独立的页面项目，内部包含对应的 HTML 及关联资源。
- `assets/`：为多个页面共用的样式、脚本或图片资源提供统一存放位置。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开，无需后端服务。
- 页面内容以信息展示和功能引导为主，不包含任何动态数据请求或第三方依赖。
- 每个页面均经过基本的兼容性测试，支持主流现代浏览器。

## 维护说明

- 本仓库采用常规 Git 工作流进行版本管理。
- 新增页面请放置在 `pages/` 目录下，并保持命名清晰。
- 如有共用资源更新，请同步修改 `assets/` 中的文件。
- 欢迎提交 Issue 或 Pull Request 进行内容修正与优化。

本仓库内容仅供学习与参考使用，不构成任何形式的服务承诺或商业用途。
